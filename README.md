## 📌 About the Project

**Vertex** is a digital school grading system designed to streamline academic record management for students, teachers, and administrators. The system provides functionality for managing grades, student profiles, subjects, and class records — similar to popular Bulgarian school platforms.

### Key Features (Planned / In Development)

- Student registration and profile management
- Grade entry, editing, and viewing by subject
- Teacher and class management
- Report generation and academic statistics
- Role-based access (Admin / Teacher / Student)

## 🚀 Getting Started

### Prerequisites

- Visual Studio (2019 or later recommended)
- Git installed on your machine

### Clone the Repository

```bash
git clone https://github.com/your-org/vertex.git
cd vertex
```

### Build & Run

1. Open the `.sln` solution file in Visual Studio
2. Set the build configuration to `Debug` or `Release`
3. Press `Ctrl + F5` to build and run

## 🔄 Git Workflow

We follow a simple branching strategy:

- `main` — stable, production-ready code
- `dev` — active development branch
- `feature/(your-feature-name)` — individual feature branches

### Commit Convention

```
[type]: short description

Types: feat, fix, docs, refactor, test
Example: feat: add grade entry for students
```
